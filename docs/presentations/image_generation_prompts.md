# Presentation Image Generation Prompts

All three assets were generated with the built-in image generation tool using
the `scientific-educational` use case. Exact technical labels were deliberately
excluded from the raster prompts and added as editable PowerPoint shapes.

## 1. System Measurement Architecture

```text
Create a clean, professional 16:9 whitepaper-style visualization of a GPU energy
microbenchmark system. Show a host CPU workstation dispatching CUDA work to a
modern GPU accelerator. Show two independent observation channels leaving the
GPU: an energy/power measurement path and a performance-counter telemetry path,
both ending at an analysis workstation. Use white, graphite, cool gray,
restrained teal, amber, and blue. Leave white space for PowerPoint labels. No
text, numbers, logos, watermark, dark background, or decorative blobs.
```

## 2. Treatment-Control Differential Method

```text
Create a clean, professional 16:9 whitepaper-style visualization of a controlled
GPU microbenchmark. Show two parallel lanes with identical base stages. The
control lane retains loop/address/register overhead but omits the target
operation. The treatment lane adds one highlighted target path. Converge both
lanes into a differential comparison node and then into operation-normalized and
byte-normalized outputs. Use a white background, blue-gray control, teal
treatment increment, and amber comparison. Leave white space for editable
equations. No text, numbers, logos, watermark, or decorative blobs.
```

## 3. NCU Validation And Audit

```text
Create a clean, professional 16:9 whitepaper-style visualization of a GPU
experiment evidence funnel. Show multiple telemetry streams from a GPU for
tensor activity, on-chip memory, cache traffic, external memory, timing, and
power stability. Route them through sequential transparent inspection gates.
Send accepted evidence to a publication-ready report and divert rejected or
incomplete evidence to a review tray. Use white, graphite, teal, amber, blue,
and restrained red. Emphasize evidence gating, not direct component-energy
sensing. Leave white space for editable labels. No text, numbers, logos, people,
watermark, or decorative blobs.
```
