# Pre-Current-Protocol Archive

이 디렉토리는 2026-07-12 address-control exact-NCU protocol 이전의 결과 설명,
시각화와 보조 분석 스크립트를 보존한다.

포함 내용:

- 2026-07-08 RTX 3090 상세 결과/백서 원문
- 과거 auxiliary 결과를 포함한 전체 방법 비교 문서
- `clocked_empty` global-memory control 기반 coefficient 시각화
- 당시 evidence matrix, sanity audit, plotting/stability scripts

이 자료는 실험 발전 과정과 historical NCU/path evidence를 재현할 때만 사용한다.
현재 component coefficient, current strict result 또는 다른 GPU의 baseline으로 인용하지
않는다. 현행 기준은 `docs/methodology/`, `docs/platforms/`와 generated finalplan package다.

2026-07-08 strict summary와 NCU evidence 그림도 이 archive에 둔다. 역사적
`results/summary/rtx3090_strict_scope_fresh_ncu_component_coefficients_20260708.md`는
재현성을 위해 이 위치의 그림을 참조하지만, 새 strict summary 생성기는 이 정적 그림을
자동 삽입하지 않는다.
